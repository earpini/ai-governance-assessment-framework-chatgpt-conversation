# Classification validation samples

Create one JSON file per source and country. Each item must retain the source record ID, text or archived-record reference, `predictedRelevant`, `humanRelevant`, reviewer, and review date.

Minimum pilot samples:

- 50 stratified OpenAlex works per country.
- 100 stratified GDELT articles per country.

The publication report must show precision by source, country, and language. Query dictionaries may be revised once after the first review round; the revision and all changed terms must be recorded before the final sample is drawn.
